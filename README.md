# Streamlit Demo — Credit Risk (Default Probability)

This app trains a simple ML model (logistic regression) on synthetic credit data and lets you score a new applicant.

## Run locally
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Deploy (Streamlit Community Cloud)
1. Push this folder to a GitHub repo.
2. Go to Streamlit Community Cloud → New app.
3. Select your repo and set **Main file** = `streamlit_app.py`.
4. Deploy.
